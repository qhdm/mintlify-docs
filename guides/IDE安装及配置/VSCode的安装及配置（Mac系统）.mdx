#### Mac

1. **使用命令行安装 Miniconda**
   - 打开终端：使用`command`+`空格`打开聚焦搜索，输入`terminal`，回车打开终端。
   - 在终端中输入以下命令：
   ```bash
    mkdir -p ~/miniconda3
    curl https://repo.anaconda.com/miniconda/Miniconda3-latest-MacOSX-arm64.sh -o ~/miniconda3/miniconda.sh
    bash ~/miniconda3/miniconda.sh -b -u -p ~/miniconda3
    rm -rf ~/miniconda3/miniconda.sh
    ```
    - 安装完成后，输入以下命令对 bash 和 zsh shell 进行初始化：
    ```bash
    ~/miniconda3/bin/conda init bash
    ~/miniconda3/bin/conda init zsh
    ```
    - 最后在终端输入以下命令，若显示版本号则表示安装成功：
    ```bash
    conda -V
    ```

2. **安装 VSCode**
   - 前往 [VSCode 官网](https://code.visualstudio.com/) 下载并安装。
   - 打开 VSCode，在扩展（Extensions）中搜索并安装 **Python** 插件。

4. **配置 VSCode 使用 Conda 环境**
   - 打开 VSCode，新建 `hello.py` 文件并进入。
   - 按下快捷键 **Cmd+Shift+P**，输入并选择 **Python: Select Interpreter**。
   - 选择 `base` 环境（通常路径里会带有 miniconda3/envs/base）。
   - 在`hello.py`文件中输入：
     ```python
     print("Hello, Python from Conda!")
     ```
   - 点击运行，若能正确输出，说明配置成功。
