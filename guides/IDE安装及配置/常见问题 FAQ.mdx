在开始学习之前，你需要在电脑上配置好 Python 环境。我们推荐使用 **VSCode** 或 **PyCharm**，任选其一即可。 

[VSCode](https://code.visualstudio.com) 是微软推出的轻量级编辑器，结合 Miniconda 等可以方便地管理 Python 版本和依赖。而 [PyCharm](https://www.jetbrains.com/pycharm/) 是 JetBrains 开发的专业 Python 集成环境，功能更强大，适合深入学习。

根据你的电脑是Mac还是Windows，你只需要在左侧四个文档中选择一个进行配置即可。

### 1. Miniconda 安装后 CMD 无法识别 `conda` 命令怎么办？
- 说明 PATH 没有配置好，请参考 Windows 部分的 **检查系统路径 (PATH)** 步骤。

### 2. VSCode 里运行时找不到解释器怎么办？
- 按 **Ctrl+Shift+P** → 输入 **Python: Select Interpreter** → 选择 `base` 环境。

### 3. 我可以不用 Miniconda，直接装 Python 吗？
- 可以，但推荐 Miniconda，因为它能方便地管理环境，不会弄乱系统自带的 Python。
